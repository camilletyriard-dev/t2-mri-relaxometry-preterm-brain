# Data

This directory holds the imaging data and cohort metadata. NIfTI volumes are not tracked in version control due to size and data governance.

## Expected layout

```
data/
├── data_adolescent/             # EPICURE preterm + term-born 19-year-old subjects
│   ├── Epicure{ID}-qt2_reg.nii.gz
│   ├── Epicure{ID}-mask1.nii.gz
│   ├── Epicure{ID}-qt2_seg1.nii.gz
│   ├── Epicure{ID}-qt2_par1.nii.gz
│   └── Epicure{ID}-qt2_par2.nii.gz
│
├── TEs.txt                      # Shared adolescent echo-time schedule
├── demo.xlsx                    # Participant demographics (subset)
├── Epicure_psychological.xlsx   # Full cohort table with neuropsych scores
├── nm_labels.xlsx               # White-matter region label definitions
├── preterm_ids.csv              # Generated by notebook 01
└── preterm_exclusions.csv       # Generated by notebook 01
```

## Access

Data from the [EPICure study](https://www.epicure.ac.uk/) and the UCL CMBI course. Contact Andrew Melbourne (a.melbourne@ucl.ac.uk) for access. Place unzipped contents here so the paths above are satisfied.
